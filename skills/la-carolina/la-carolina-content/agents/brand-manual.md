### Elementos del logo
- **Forma C:** Gran letra C curva que actúa como marco identitario.
- **Figura del conductor/persona:** Cabeza circular y trazo humano integrado a la C.
- **Texto "CAROLINA":** Tipografía gruesa personalizada; puede ir dorado, negro o blanco según fondo.
- **Subline "TRANSPORTE CON CORAZÓN":** Slogan institucional bajo la palabra CAROLINA.
- **Subrayado:** Línea horizontal roja/dorada/negra según variante.

### Archivos disponibles
- `logo-principal-rojo-dorado-negro.jpg` — principal sobre fondo blanco: símbolo rojo, CAROLINA dorado, slogan negro.
- `logo-principal-fondo-blanco-centrado.jpg` — principal centrado con aire blanco.
- `logo-principal-transparente.png` — versión PNG con fondo transparente para HTML, portadas y composiciones limpias.
- `logo-principal-fondo-negro.jpg` — versión optimizada sobre fondo negro/oscuro.
- `logo-rojo-negro.jpg` — símbolo rojo + texto negro.
- `logo-monocromo-negro.jpg` — versión negra monocromática.
- `logo-simbolo-dorado-slogan.jpg` — versión/símbolo dorado con slogan.
- `simbolo-c-dorado-fondo-negro.jpg` — símbolo C dorado sobre negro.
- `simbolo-c-dorado-fondo-rojo.jpg` — símbolo C dorado sobre rojo.

### Variantes cromáticas aprobadas por fondo
1. **Fondo blanco/claro:** usar logo principal rojo + dorado + negro.
2. **Fondo dorado:** usar símbolo rojo + texto negro + slogan/subrayado rojo.
3. **Fondo rojo:** usar versión blanca/dorada para contraste.
4. **Fondo negro:** usar versión fondo negro o logo rojo/dorado con slogan blanco; evitar recuadro blanco si existe PNG/versión oscura.
5. **Sobre fotografía:** usar logo blanco o versión de alto contraste; oscurecer la foto con overlay negro para legibilidad.
6. **Aplicación monocromática:** usar `logo-monocromo-negro.jpg` solo para documentos de una tinta, sellos, marcas de agua o aplicaciones formales.

### Reglas de posicionamiento
- **Portadas digitales:** logo visible arriba o abajo, preferiblemente versión transparente/oscura integrada al fondo.
- **HTML/Netlify:** verificar que el logo cargue con HTTP 200 y que no quede encerrado accidentalmente en un recuadro blanco.
- **Membrete/documentos:** logo principal sobre fondo blanco en esquina superior izquierda.
- **Presentaciones:** usar versión oscura en portadas negras y versión principal sobre slides claros.
- **Fotografías:** aplicar overlay oscuro antes de poner logo blanco o de alto contraste.
